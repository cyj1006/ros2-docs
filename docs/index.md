# ROS2 新手指导

欢迎来到 **ROS2 新生完全学习指导书**。

这是一本面向 **完全零基础** 的 ROS2 入门指南，目标是：

- 帮你快速理解 ROS2 是什么、能做什么
- 手把手带你完成环境搭建
- 跑通第一个 ROS2 程序（Hello / Pub-Sub / turtlesim）
- 少踩坑，少走弯路

---

## 适合谁？
- 从没接触过 ROS / ROS2 的新生
- 只会一点 Python / C++，但想做机器人
- 想尽快“跑起来”，而不是只看概念的人

---

## 推荐学习路径
1. 了解 ROS2 是什么（概念）
2. 安装 Ubuntu + ROS2（环境）
3. 跑通第一个节点（Hello World）
4. 理解节点 / 话题 / 通信
5. 做一个能动的小项目（turtlesim）

👉 左侧 **Chapters** 按顺序阅读即可。
